Framaterial Framework

Framaterial is an open-source framework that allows you to create projects with the material design elements and rules

It's composed of many components and javascript functions to make it easy to use directly out of the box

With Framaterial, you'll be able to create single page apps & websites, to complex apps & websites, without the help of any thirs pary libraries, frameworks or plugins.

It's have been thought to make its integration easy for everyone, with a logic and semantic structure

In order to use it, download the Framaterial.zip, and expand it at the root of your project.
